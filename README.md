# RunConnect

A web-based application designed to enhance the running experience for individuals and groups in Hamilton, Ontario. This platform offers a comprehensive and interactive online space where runners can plan routes, connect with other runners, and track their progress.

## Features

- Route planning and exploration
- Community engagement and group running
- Personal statistics and progress tracking
- Weather and safety alerts
- Interactive dashboards
- Mobile-first responsive design

## Tech Stack

- Frontend: React with TypeScript (chosen for component-based architecture and mobile-first capabilities)
- Backend: Laravel (using raw SQL queries, no Eloquent ORM)
- Styling: TailwindCSS for responsive design
- Database: MySQl for SQL-based relational database 
- Authentication: Laravel's authentication system with custom SQL implementation
- Maps Integration: (TBD - Mapbox/Leaflet)
